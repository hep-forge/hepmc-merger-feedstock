Feedstock for [hepmc-merger](https://github.com/eic/HEPMC_Merger) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Merges signal and background events from HepMC3 files

## Install

```bash
conda install -c hep-forge -c conda-forge hepmc-merger
```
